# Программа для решения математической задачи - код Прюфера

Программа решает математический метод - код Прюфера. **Код Прюфера** – это способ взаимно однозначного кодирования помеченных деревьев с n вершинами с помощью последовательности n-2 целых чисел в отрезке [1,n]. То есть, можно сказать, что код Прюфера – это биекция между всеми остовными деревьями полного графа и числовыми последовательностями.

## Начало работы с программой для решения математической задачи

1. Для запуска проекта необходимо создать текстовый файл, в котором будут переданы первичные данные для решения задачи код Прюфера;
2. После того как Вы создали текстовый файл, Вам необходимо открыть программу и нажать клавишу F5 для запуска отладки;
3. На консоли Вы увидите сообщение "Введите путь до файла:";
4. В консоли вводите путь до созданного Вами файла из пункта 1.
